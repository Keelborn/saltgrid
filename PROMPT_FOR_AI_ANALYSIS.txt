- Что может сломаться
- Рекомендация по фиксу

## 3. ОПТИМИЗАЦИЯ ПРОИЗВОДИТЕЛЬНОСТИ
Критически важный раздел. Этот мод — чанк-генератор, он вызывается МИЛЛИОНАМИ раз.
Ищи:
- Лишние вычисления (одно и то же считается N раз за чанк)
- Аллокации объектов в hot path (especially double[] и int[])
- Дублирование кода между fillFromNoise / getBaseHeight / getBaseColumn
- Вызовы gridIslandH / islandH / islandDist которые пересчитывают одно и то же
- Массивы/коллекции которые можно кэшировать или избежать
- N+1 проблемы (9-ячейковый цикл для каждого блока)
- Неэффективные хеш-функции
- Лишние sqrt/pow вызовы

## 4. БАГИ ГЕНЕРАЦИИ МИРА (worldgen glitches)
Ищи визуальные артефакты:
- Острова без рельефа или с неправильным рельефом
- Блоки которые не ставятся / ставятся неправильно
- Дыры в генерации (air pockets под землёй)
- Вода которая не заполняет правильные слои
- Декорации которые не генерируются или генерируются в воздухе
- Складки/ступеньки на береговой линии
- Проблемы на границах чанков (seams)
- Телепорт /island на неправильные координаты

## 5. ЛОГИЧЕСКИЕ ОШИБКИ В КОДЕ
- Неправильные формулы
- Off-by-one ошибки
- Мёртвый код
- Неиспользуемые переменные
- Неправильные типы данных (int overflow при больших координатах)
- Отсутствующие проверки (NPE, ArrayIndexOutOfBounds)

## 6. ПРОБЛЕМЫ С РЕГИСТРАЦИЕЙ И РЕСУРСАМИ
- Не совпадают blockstate/модель/текстура
- Неправильные render_type (cutout vs solid)
- Отсутствующие файлы
- Неправильные пути в JSON

ДОПОЛНИТЕЛЬНЫЙ КОНТЕКСТ О ПРОЕКТЕ:
- NeoForge 1.21.1, Java 21
- Кастомный измерение "ocean" с уникальным чанк-генератором
- Два типа островов: спавн-остров (уникальная форма) и grid-острова (по сетке 2048x2048)
- Кастомные блоки: palm (направленный), palm_leaf, shell (3 вариации), ground_decoration (11 типов)
- Генераторы деревьев: PalmGenerator (3 варианта пальм), RainTreeGenerator (3 варианта дождевых деревьев)
- Пальмы и деревья генерируются через хардкод координат (NBT templates не работают из кастомного ChunkGenerator)
- Известный баг: чёрные текстуры на блоке palm (два half-блока окрашены в чёрный вместо текстуры дерева)
