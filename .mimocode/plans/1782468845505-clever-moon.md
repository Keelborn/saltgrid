# План: Генерация островов (grid-based + рандомизация)

## Концепция
Мир делится на квадраты 2048x2048. Внутри каждого — зона 512x512, рандомная точка = центр острова. Размер сильно варьируется (50-300 блоков). Некоторые ячейки пустые (hash < порога). Поверхность — песок.

## Файлы
- `src/main/java/com/jokerdayn/swworldgencore/worldgen/OceanChunkGenerator.java` — единственный файл для изменений

## Шаг 1: Добавить `islandHeight(x, z)`

Новый метод. Вычисляет вклад острова в высоту в точке (x, z).

```java
private double islandHeight(int x, int z) {
    int cellX = Math.floorDiv(x, 2048), cellZ = Math.floorDiv(z, 2048);

    // ~60% ячеек содержат остров, 40% — пустой океан
    if (hash(cellX * 11, cellZ * 13) > 0.6) return 0;

    double cx = cellX * 2048 + 768 + hash(cellX * 2, cellZ * 2) * 512;
    double cz = cellZ * 2048 + 768 + hash(cellX * 2 + 1, cellZ * 2 + 1) * 512;

    // радиус 25-150 блоков (диаметр 50-300) — сильная вариация
    double radius = 25 + hash(cellX * 3, cellZ * 3) * 125;

    double dx = x - cx, dz = z - cz;
    double t = Math.sqrt(dx * dx + dz * dz) / radius;
    if (t >= 1) return 0;

    double falloff = (1 - t * t) * (1 - t * t);

    // пик пропорционален размеру: чем больше остров — тем выше
    double peak = 10 + radius * 0.15 + hash(cellX * 5, cellZ * 7) * 10;
    return falloff * peak;
}
```

## Шаг 2: Добавить `finalHeight(x, z)`

Объединяет океанское дно и остров. Заменяет все вызовы `floorAt` снаружи самого метода `floorAt`.

```java
private int finalHeight(int x, int z) {
    int ocean = floorAt(x, z);
    int island = (int)(seaLevel + islandHeight(x, z));
    return Math.max(ocean, island);
}
```

## Шаг 3: Обновить `pickSurface`

Когда `islandHeight > 0` — поверхность песок (берег/остров). Подводная логика без изменений.

```java
private BlockState pickSurface(int x, int z, int fl) {
    double ih = islandHeight(x, z);
    if (ih > 5) return Blocks.SAND.defaultBlockState();      // сухая часть острова
    if (ih > 0) {                                             // берег — 70% песок, 30% гравий
        return hash(x * 7, z * 13) < 0.7
                ? Blocks.SAND.defaultBlockState()
                : Blocks.GRAVEL.defaultBlockState();
    }
    // ... существующая подводная логика без изменений
}
```
