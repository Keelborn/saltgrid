# План: Плавный переход океан → остров

## Проблема

Сейчас `computeFloor()` имеет жёсткую ветку: `if (spRaw > 0) → island height`, `else → ocean floor`. Когда `spRaw` падает до 0 (на краю пера), пол внезапно прыгает с `seaLevel` (63) на `oceanFloor` (seaLevel-4 = 59 или ниже). Результат — подводный обрыв вместо плавного склона.

Блоки тоже меняются резко: песок → гравий/галька без промежуточной зоны.

## Что нужно

1. **Shelf zone** — пол постепенно поднимается от ocean floor к уровню моря по мере приближения к острову
2. **Блочный переход** — песок доминирует у берега, дальше плавно переходит в океанское дно
3. **Нет обрывов** — высота плавная, без скачков

## Зоны (по `t = islandDist`)

| Зона | t | Высота пола | Поверхность | Вода |
|------|---|-------------|-------------|------|
| Остров | < 1.0 | seaLevel + spRaw | песок/трава | нет |
| Пляж | 0.95–1.0 | ≈ seaLevel | песок | 0–1 блок |
| Шельф | 1.0–1.35 | плавно ↓ к oceanFloor | песок → смесь | 1–5 блоков |
| Глубокий океан | > 1.35 | oceanFloor | смесь | 4+ блоков |

### 1. `computeFloor()` — добавить плавный шельф

**Сейчас** (строки 208–224):
```java
private int computeFloor(int x, int z, double spRaw) {
    // ... ocean floor noise ...
    if (spRaw > 0.0) {
        return Math.max(oceanFloor, seaLevel + (int) Math.round(spRaw));
    }
    return oceanFloor;
}
```

**Нужно**: передавать `t` (islandDist) и добавить промежуточную зону:

```java
private int computeFloor(int x, int z, double t, double spRaw) {
    // ... ocean floor noise (без изменений) ...
    int oceanFloor = Math.max(-63, Math.min((int) h, seaLevel - 4));

    if (spRaw > 0.0) {
        return Math.max(oceanFloor, seaLevel + (int) Math.round(spRaw));
    }

    // Шельф: t ∈ [1.0, maxT] — пол поднимается к морю
    double maxT = 1.0 + SPAWN_ISLAND_FEATHER / SPAWN_ISLAND_RADIUS;
    if (t < maxT) {
        double norm = (t - 1.0) / (maxT - 1.0); // 0 у берега, 1 у края
        double blend = norm * norm * (3.0 - 2.0 * norm); // smoothstep
        int target = seaLevel - 1;
        return (int) Math.round(oceanFloor + (target - oceanFloor) * (1.0 - blend));
    }

    return oceanFloor;
}
```

### 2. `pickSurf()` — блочный переход через `t`

**Сейчас** (строки 230–238):
```java
private BlockState pickSurf(int x, int z, int fl, double spRaw) {
    if (fl < seaLevel) return oceanFloor(x, z, fl);
    if (spRaw < 4.0) return SAND;
    return GRASS_BLOCK;
}
```

**Нужно**:
```java
private BlockState pickSurf(int x, int z, int fl, double t, double spRaw) {
    if (spRaw > 4.0) return GRASS_BLOCK;

    double maxT = 1.0 + SPAWN_ISLAND_FEATHER / SPAWN_ISLAND_RADIUS;

    // Шельф: песок плавно переходит в океанское дно
    if (t >= 1.0 && t < maxT) {
        double norm = (t - 1.0) / (maxT - 1.0);
        // У берега — чистый песок. Дальше — вероятность гравия растёт
        if (hsh(x * 7, z * 13) > norm * 0.4) return SAND;
        return oceanFloor(x, z, fl);
    }

    if (fl < seaLevel) return oceanFloor(x, z, fl);
    return SAND;
}
```

### 3. `subSurf()` — подповерхность шельфа

**Сейчас** (строки 258–262):
```java
private BlockState subSurf(int x, int z, int fl, double spRaw) {
    if (fl < seaLevel) return null; // stone
    if (spRaw < 4.0) return SAND;
    return DIRT;
}
```

**Нужно**:
```java
private BlockState subSurf(int x, int z, int fl, double t, double spRaw) {
    if (fl >= seaLevel) {
        return spRaw < 4.0 ? SAND.defaultBlockState() : DIRT.defaultBlockState();
    }
    // Шельф — подповерхность из песка/гравия вместо голого камня
    double maxT = 1.0 + SPAWN_ISLAND_FEATHER / SPAWN_ISLAND_RADIUS;
    if (t < maxT) {
        return hsh(x * 11, z * 17) < 0.6 ? SAND.defaultBlockState() : GRAVEL.defaultBlockState();
    }
    return null; // stone
}
```
